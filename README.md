Spots is an image sharing site.

### Overview

- Intro

- Deployed Project on GitHub Pages

**Intro**

This project is made so all the elements are displayed correctly on popular screen sizes. Spots is an image sharing site that allows users to upload images, caption posts, and create a profile that highlights each user's individuality. Through the use of HTML and CSS elements, this site is responsive to fit various screen sizes and is syled to be visually appealing and easy to use.

**Technologies and Techniques**

- HTML
- CSS
- Flexbox
- Grid
- Responsive Design
- Media Queries
